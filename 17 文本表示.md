
## one hot representation
词典：[我们，去，爬山，今天，你们，昨天，跑步]
词典中每个单词是有顺序的
向量长度等于词典长度
每个单词的表示：
我们：(1, 0, 0, 0, 0, 0, 0)
爬山：(0, 0, 1, 0, 0, 0, 0)
跑步：(0, 0, 0, 0, 0, 0, 1)
昨天：(0, 0, 0, 0, 0, 1, 0)

### 句子的表示
1. boolean表示

词典：[我们，又，去，爬山，今天，你们，昨天，跑步]
我们 今天 去 爬山
(1, 0, 1, 1, 1, 0, 0, 0)
你们 昨天 跑步
(0, 0, 0, 0, 0, 1, 1, 1)
你们 又 去 爬山 又 去 跑步
(0, 1, 1, 1, 0, 1, 0, 1)

2. count base 表示

词典：[我们，又，去，爬山，今天，你们，昨天，跑步]
我们 今天 去 爬山
(1, 0, 1, 1, 1, 0, 0, 0)
你们 昨天 跑步
(0, 0, 0, 0, 0, 1, 1, 1)
你们 又 去 爬山 又 去 跑步
(0, 2, 2, 1, 0, 1, 0, 1)

缺陷：
* 并不是出现越多越重要
* 也不是出现越少越不重要

3. tf-idf 文本表示方法
定义：`tfidf(w) = tf(d,w) * idf(w)`
w: 一个单词
tf(d,w): 文档d中w的词频，就是count base的表示
idf(w): log N/N(w)，N：语料库中的文档总数，N(w)：词语w出现在多少个文档中，用于考虑单词的重要性：有些单词经常在很多句子和文章中都出现，这种单词的重要性一般比较弱，而有些单词一般不出现，反而出现在了这个句子中，这种单词的重要性一般比较高，log是为了不希望这个值太大

s1: 今天 上 NLP 课程
S2: 今天 的 课程 有 意思
s3: 数据 课程 也 有 意思
词典：[今天 上 NLP 课程 的 有 意思 数据 也]
s1: (1 * log 3/2, 1 * log 3/1, 1 * log 3/1, 1 * log 3/3, 0, 0, 0, 0, 0)
