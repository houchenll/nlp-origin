
计算两个句子的相似度

## 欧式距离 d = |s1 - s2|
s1: 我们 今天 去 爬山 = (1,0,1,1,0,0,0,0)
s2: 你们 昨天 跑步 = (0,0,0,0,0,1,1,1)
s3: 你们 又 去 爬山 又 去 跑步 = (0,2,2,1,0,1,0,1)

s1 = (x1, x2, x3)
s2 = (y1, y2, y3)
d = sqrt( (x1-y1)^2 + (x2-y2)^2 + (x3-y3)^2 )

缺点：
向量是有方向的，但是欧式距离没有考虑向量方向


## 余弦相似度 d = s1 * s2 / (|s1| * |s2|)
s1 * s2 表示内积，考虑了方向
|s1|表示向量的范式长度， normalization
d = x1*y1 + x2*y2 + x3*y3 / ( sqrt(x1^2 + x2^2 + x3^2) * sqrt(y1^2 + y2^2 + y3^2) )
余弦相似度值越大，相似度越大，值越小，相似度越小
