
# Sampling methods

Importance sample, negative sample, NCE, ...  
从服从某一分布的样本集中，采样，取出一些样本。来估计样本的分布，或做一些生成的任务。

## 为什么需要采样
现实生活中，大多数数据都是非常庞大的，总体分布可能包含无数多的样本点，模型无法对这些海量数据直接建模，而且效率上也不推荐。

因此，我们一般会从总体样本上抽取出一个子集来近似总体分布，这个子集被称为训练集。然后模型训练的目的是最小化训练集上的损失函数，训练完成后，需要另一个数据集来评估模型，称为测试集。

采样是一种近似，近似我们的期望。期望一部分是我的统计量信息，另外能得到后验推断，推断更重要，而且可以可视化。  

## 采样方法
逆变器采样、拒绝采样(接受采样)、重要性采样(Importance Sampling)、马尔科夫蒙特卡洛采样法&积分

### 蒙特卡罗方法
定义：通过大量重复试验，通过统计频率，来估计概率，从而得到某个问题的求解   
例：正方形内有一个圆，已知正方形面积计算分式，未知圆面积计算公式。方法：在正方形随机均余撒n个点，落在圆内点的个数为k个，那么圆的面积=`正方形面积*k/n`  
随机模拟

### 伯努利大数定理
定义：设fa是n次独立重复试验中事件A发生的次数，P是事件A在每次试验中发生的频率，对任意的x>0，有lim P(|fa/n - p| < x) = 1，其中p表示事件A发生的概率  

### 辛钦大数定理
设X1, X2, ...相互独立，服从同一分布的随机变量序列，且具有数学期望E[Xk] - µ (k = 1,2,3,...)，做设前n个变量的算术平均`1/n ∑Xk`，则对于任意∂>0，有`lim P(|1/n ∑Xk - µ| < ∂) = 1`
