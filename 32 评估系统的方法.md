
## 评估方法有哪些
准确率、精确率、召回率  

## 准确率(accuracy)
定义：正确分类的样本个数/总样本个数  
缺陷：当样本不平衡时，用准确率不合适。如果正样本和负样本数量相差很大，此时叫做样本不平衡。如1000个人样本中，只有5个人是癌症患者，我不做任何训练，就预测所有样本是非癌症患者，这样的算法的准确率高达99.5%，但并没有什么用。  

## 精确率(precision)
如1000个人样本中，有10个人是癌症患者.
算法判断10人为癌症，其中有8人确实是癌症患者，那么算法准确率为：8/10  
负样本准确率：988/990  
用P表达  

## 召回率(recall)
1000个人样本中，有10个人是癌症患者，算法判断出其中8个人是癌症患者，那么召回率为：8/10
用R表达  

## 例子1
用T表达正样本，N表示负样本  
真实数据为：N N N T T N N N N T  
预测结果为：N N T T N N N N N T  
正样本准确率为：P=2/3  
负样本召回率为：R=2/3  

## 例子2
真实数据为：N N N T T N N N N T  
预测结果为：N T T T T N N T N T  
正样本准确率为：P=3/6  
负样本召回率为：R=3/3  

## 准确率和召回率的关系
想要提升召回率的时候，相当于在牺牲精确率  
P和R之间有互斥的关系，不可能同时要求很高的准确率和召回率  
一般会定义一个阈值M，当P>M && R>M时，系统满足要求  

## F1 Score
用一个数字评估AI系统，是精确率和召回率的综合  
f1-score = 2 * presicion * recall / (percision + recall)  
当有多个分类时，分别计算f1-score，然后计算它们的算术平均
