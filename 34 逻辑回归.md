
# 逻辑回归
经典二分类模型  
面试必考  
工业界用的最多的模型  

根据样本，学习输入X到输出Y之间的映射ƒ: X -> Y  
即：P(Y|X)  
对二分类问题，P(Y1|X) > P(Y2|X)即判断分类为Y1  
P(Y|X)不能用wx+b这种线性表示，因为线性表示wx+b落在实数空间范围，不满足概率的性质  


## 应用场景
贷款违约情况（会违约、不会违约）
广告点击问题（会点击、不会点击）
商品推荐（会购买、不会购买）
情感分析（正面、负面）
疾病诊断（阳性、阴性）


## 逻辑函数 Logistic Function/sigmoid
逻辑函数通常用`σ(x)`表示

```
y = 1 / 1 + e^(-x)
```

函数性质：
y: (0,1)  
x: R  
x = 0时y=0.5  
x ≥ 6 时，y趋近于1  
x ≤ -6 时，y趋近于0  


## 逻辑回归条件概率
P(Y|X;w)= σ(wx+b) => (0,1)  
w是矩阵  
b是bias纠正数  


## 逻辑回归是线性分类器？
线性函数外面叠加非线性的逻辑函数，那么结合起来的逻辑回归是线性函数吗？  
答案：逻辑回归是线性分类器  


## 目标函数
p(y|x,w)=p(y=1|x,w)^y[1-p(y=1|x,w)]^(1-y)
最大化目标函数，最大似然：
wmle,bmle=argmaxw连乘p(yi|xi,w,b)
优化算法(随机梯度下降, adam, ...)  


## 梯度下降法
求使得ƒ(w)值最小的参数w  
* 是否凸函数，逻辑回归是凸函数
* 最优化算法：梯度下降法、sgd、adagrad、adam

步长可以根据时间变化而变化，如随着时间增长，步长变小  
停止条件：  
1. 计算w时，不断计算目标函数，当目标函数不再下降，或下降幅度小于一定数值量，如0.0001  
2. 固定循环次数


## 随机梯度下降法
梯度下降中，n表示样本的个数，有时是几十亿的数量级，每次更新都需要计算所有样本的梯度，时间复杂度太大。  
随机梯度下降法，首先把所有的数据打乱，做一次新的排列，一个样本一个样本地考虑，对每个样本，分别用梯度下降法去更新梯度  
一般情况下，使用随机梯度下降，替代梯度下降，使得我们能够在最短的时间内拿到最优解  

两个梯度下降法折中的方法，叫做mini-batch graient descent   


## 线性可分
定义：存在一条直线，可以把点完美地分开  
如果线性可分，可能会存在无数条直线可以把点分开  
当给定的数据线性可分的时候，逻辑回归的参数会趋向于无穷大  

过拟合
当w很大时，这是过拟合现象，需要避免


## 正则 Regularzation
为了避免w太大，加入正则化项`L2 Norm = w1^2 + w2^2 + ... + wd^2`  
新的目标函数 = 原目标函数 + λ * L2 Norm  
λ是超参数，控制正则项的权重，起到平衡作用
