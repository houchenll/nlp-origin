
## 模型和目标函数的关系
在一个应用场景上，选择一个算法模型，一旦选择了算法模型，就选择了对应的目标函数。接下来，在训练数据集上求目标函数的最优解，这是优化过程。  
有时，根据人的经验，可以人工地预先设置某些变量的范围。  

应用场景 => 定模型 => 定含有待定参数的目标函数 => 优化

模型可以理解为类似于类的抽象的定义，目的应用场景时选用模型时，类似于实例化模型(类比实例化类)

特征
  * 特征工程
    * 特征编码
      * encoding
    * 特征预处理
      * 归一化
      * 异常
      * noisy value
