
# P vs NP vs NP Hard vs NP Complete

问题 -> 得到最好的方法 --> 得到最好方法的时间复杂度

时间复杂度可能是O(p^n)或O(n^p)，p是常数
O(p^n)是指数级复杂度
O(n^p)是多项式复杂度

多项式复杂度是可以解决的问题
指数级复杂度是不可以解决的问题

指数级复杂度的问题
对于小型的问题，n比较小，仍然可以解决
对于大型问题，n比较大，可以提供一个多项式复杂度的近似算法，不保证得到精确结果

P问题：可以在多项式的复杂度内解决
NP：在多项式复杂度内能够verify的，是一个验证问题，不用解决
NP Hard：在有限的时间内解决不了的问题
NP Complete：属于NP Hard，在有限的时间内解决不了的问题
