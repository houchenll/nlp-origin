## 安装环境

```
mkvirtualenv spider
workon
workon spider

pip install Scrapy
pip install mysqlclient

# 创建项目
cd /Users/ll/workspace/python/spider/project
scrapy startproject my_spider
cd my_spider
scrapy genspider qizha https://tieba.baidu.com/f\?ie\=utf-8\&kw\=%E9%98%B2%E6%AC%BA%E8%AF%88\&fr\=search
```
